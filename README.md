# KubeKloud Theme

Faça um clone do repositorio:
```bash
git clone 
```

Agora é só executar esse comando:
```bash
mkdir /etc/kubekloud-theme/ && \
  mv ~/Kubekloud-theme/kubekloud.bashrc.sh /etc/kubekloud-theme/ && \
  echo "source /etc/kubekloud-theme/kubekloud.bash.sh" >> ~/.bashrc && \
  cd ~ && source .bashrc
```
